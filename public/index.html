Up and running in 3 steps

Install once — from then on the app updates itself automatically, no re-downloading.

Download the installer

Download PriorityTeamConnect-Setup.exe and double-click it. It installs in seconds — no admin needed.

Run it

Windows may warn once (it's unsigned): click More info → Run anyway. Future updates apply silently in the background.

Create your account

Click Create an account, sign in, and check Keep me signed in. Your administrator will assign your role.
